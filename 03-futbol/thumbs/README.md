# Miniaturas (previews) del index

Cada tarjeta del `index.html` muestra arriba una miniatura del gráfico. La
imagen se busca en esta carpeta con el nombre del gráfico. Si el archivo no
existe, la tarjeta queda sin imagen (no se rompe).

Dejá acá los PNG con **estos nombres exactos** (el formato cuadrado del botón
"Descargar PNG" va perfecto; se recorta solo con CSS a 150px de alto):

| Gráfico | Archivo |
|---|---|
| 1 — Sudamérica juega en otra liga        | `chart-elo-pib.png` |
| 2 — Talento ajustado por población       | `chart-talento.png` |
| 3 — Dónde nació el fútbol moderno (mapa) | `chart-clubage-map.png` |
| 4 — Clubes antiguos y talento monopolizado | `chart-talento-clubes.png` |
| 5 — Trayectorias de las selecciones      | `chart-elo-trayectoria.png` |
| 6 — Natividad de los mundialistas        | `chart-natividad.png` |
| 7 — Ligas de destino                     | `chart-ligas.png` |
| 8 — De dónde salen los mundialistas (mapa) | `chart-birthplace.png` |

Tip: no hace falta que sean chicos; el CSS los encuadra. Pero si querés que el
repo no pese, redimensionalos a ~600 px de ancho.
